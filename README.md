# S V Balaji Banquet Halls – V2 (Devotional x Gen Z)

A hyper-immersive, addictive web experience blending South Indian Balaji traditions (kumkum, pasupu, marigold, silk, rangoli) with contemporary motion and UX.

## Highlights
- Divine hero with floating diya embers and petals (canvas particles)
- Kumkum–Pasupu gradient brand system
- Temple-inspired UI (marigold overlays, silk glassmorphism)
- Heritage, Venues, Ritual Services, Gallery, Blessings, Booking
- Mobile-first, accessible, motion-safe fallbacks

## Run locally
```bash
# From project root
python3 -m http.server 8000
# open http://localhost:8000
```

## Customize
- Update images in `index.html` <img> tags
- Edit palette in `:root` within `styles.css`
- Tweak particle counts in `script.js` → `initDivineParticles()`
